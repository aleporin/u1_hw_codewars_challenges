## Return Negative

```js
function makeNegative(num) {
  return num < 0 ? num : -num;
}
```

## Sum of Positive

```js
function postiveSum(arr) {
let sum = 0
  arr.forEach(num => num > 0 && (sum += num))
  return sum}
```
